# DevelopsToday

## Backend Setup and Launch

1. Navigate to the backend directory:

```bash
cd DevelopsToday-BE
```

2. Install dependencies:

```bash
npm install --legacy-peer-deps
```

3. Start the server:

```bash
npm run dev
```

## Frontend Setup and Launch

1. Navigate to the frontend directory:

```bash
cd DevelopsToday-FE
```

2. Install dependencies:

```bash
npm install --legacy-peer-deps
```

3. Start the server:

```bash
npm run dev
```
